These fields shape future generation. Existing entries stay as-saved; the changes apply from the next turn forward.
  • mode — Reframes the user's relationship to the story going forward. New generation will treat you as a character instead of a director.
  • narration — Changes the AI's prose voice from the next turn forward. Existing entries are unchanged.
  • composerWrapPov — Affects how new user input in Do / Say / Think modes is rendered. Existing entries stay as-saved.
  • activePackId — Switches the active pack — the prompt shape rewrites on the next turn. Story content stays; generation patterns change.
